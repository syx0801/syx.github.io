DP专题 关路灯

传送门

题目描述

某一村庄在一条路线上安装了n盏路灯，每盏灯的功率有大有小（即同一段时间内消耗的电量有多有少）。老张就住在这条路中间某一路灯旁，他有一项工作就是每天早上天亮时一盏一盏地关掉这些路灯。

为了给村里节省电费，老张记录下了每盏路灯的位置和功率，他每次关灯时也都是尽快地去关，但是老张不知道怎样去关灯才能够最节省电。他每天都是在天亮时首先关掉自己所处位置的路灯，然后可以向左也可以向右去关灯。开始他以为先算一下左边路灯的总功率再算一下右边路灯的总功率，然后选择先关掉功率大的一边，再回过头来关掉另一边的路灯，而事实并非如此，因为在关的过程中适当地调头有可能会更省一些。

现在已知老张走的速度为1m/s，每个路灯的位置（是一个整数，即距路线起点的距离，单位：m）、功率（W），老张关灯所用的时间很短而可以忽略不计。

请你为老张编一程序来安排关灯的顺序，使从老张开始关灯时刻算起所有灯消耗电最少（灯关掉后便不再消耗电了）。

Input

文件第一行是两个数字n(1<=n<=50，表示路灯的总数)和c(1<＝c<=n老张所处位置的路灯号)；

接下来n行，每行两个数据，表示第1盏到第n盏路灯的位置和功率。数据保证路灯位置单调递增。

Output

一个数据，即最少的功耗(单位：J，1J＝1W·s)。

解题思路

dp[i][j] 表示区间[i,j]灯全部关闭这一时间点消耗的总功耗，

还要考虑一点，关掉区间[i,j]的灯时，人站在哪一端？

所以还要数组还要加一维，用0和1来表示人此时在左端和右端，

dp[i][j][0]表示区间[i,j]灯全部关闭并且人在左端，此时消耗的总功耗，
dp[i][j][1]表示区间[i,j]灯全部关闭并且人在右端，此时消耗的总功耗。

尝试写出状态转移方程

其中，calc()函数计算从上一个区间转移过来时没关的路灯消耗的电力

再来搞一下calc()函数

功耗等于时间乘开着灯的总功率，人的速度为1m/s，所以从i走到j时间在数值上等于两地之间的距离loc[i]-loc[j]，现在还开着的灯的总功率怎么求？
n个灯的总功率减去区间[i][j]灯的总功率，所以很容易想到用前缀和处理。
